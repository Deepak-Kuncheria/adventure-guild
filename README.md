## Adventure guild

Scripts to know:

- For running server

```bash
npm run dev
```

- For Drizzle ORM

```bash
npm run db_generate //To generate db migration files

npm run db_migrate //To push the migration files to global db
```

- For running Playwright tests

```bash
npm run tests //To run all tests

npm run tests_ui_mode //To run all tests in Playwright's UI mode
```
